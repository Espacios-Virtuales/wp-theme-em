# Escuela Mística - Plataforma Web

Este repositorio contiene el desarrollo de la plataforma web de **Escuela Mística**, diseñada para ofrecer herramientas de autoconocimiento, autosanación y despertar espiritual. El proyecto incluye funcionalidades como:

- **Gestión de contenido dinámico**: Shortcodes personalizados para recursos gratuitos, testimonios, contacto y más.
- **Integraciones externas**: YouTube, Calendly, redes sociales y otros servicios.
- **Diseño visual único**: Interfaz alineada con la identidad de Escuela Mística, utilizando colores y estilos personalizados.
- **Optimización técnica**: Código modular y reutilizable, con estándares de desarrollo en WordPress.

## Tecnologías Utilizadas

- **WordPress**: CMS base para la plataforma.
- **SCSS**: Para estilos personalizados y estructura modular.
- **PHP**: Desarrollo de funciones y shortcodes.
- **HTML5/CSS3**: Diseño frontend.
- **Bootstrap 5**: Framework para diseño responsivo.

## Instrucciones de Uso

1. Clona este repositorio en tu servidor local o remoto:
   git clone https://github.com/usuario/escuela-mistica.git
2. Configura WordPress e instala los plugins necesarios.
3. Importa los datos de prueba si es necesario.
4. Personaliza los ajustes y contenidos desde el administrador de WordPress.
